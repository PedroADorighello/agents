# AI Agents Repository 🤖

Este repositório centraliza o desenvolvimento e a implementação de diversos **Agentes de IA** especializados. O objetivo é explorar diferentes arquiteturas de agentes autônomos e semi-autônomos, utilizando modelos de linguagem de larga escala (LLMs) para automatizar tarefas, gerenciar fluxos de trabalho e integrar serviços externos.

## 📌 Sobre o Projeto

Os agentes aqui contidos variam de simples assistentes de conversação a sistemas complexos capazes de realizar chamadas de função (function calling), navegar na web e interagir com APIs de terceiros.

## 🚀 Agentes Disponíveis

Atualmente, o repositório conta com:
- **Trello Task Manager**: Um agente focado em produtividade que gerencia quadros e cartões no Trello via linguagem natural.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python
- **Frameworks:** LangChain
- **Modelos:** Google Gemini
- **Integrações:** APIs REST e SDKs de serviços cloud.
